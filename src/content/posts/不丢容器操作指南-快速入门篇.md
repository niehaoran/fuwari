# 不丢容器快速入门指南

## 什么是不丢容器？能解决什么问题？

### 核心功能
不丢容器是一个基于 Kubernetes 的**云原生容器托管平台**，让你的应用轻松上云！

- **一键公网部署** - 在线托管 Docker 镜像，瞬间获得公网访问能力
- **在线构建** - 基于 GitHub 的代码构建，生成 Docker 镜像
- **镜像加速服务** - 解决国内 Docker 镜像拉取慢、失败等痛点
- **容器编排** - 提供服务发现、负载均衡、监控等功能
